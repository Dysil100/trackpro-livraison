# TrackPro — Suivi des Livraisons

Application web de gestion et tracking de livraisons. Stack : Node.js + SQLite (aucune dépendance externe).

## Installation (1 étape)

```bash
git clone https://github.com/Dysil100/trackpro-livraison
cd trackpro-livraison
npm install && npm start
```

Ouvrir **http://localhost:3000**

> La base de données et les données de démo sont créées automatiquement au premier lancement.
